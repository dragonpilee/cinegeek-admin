CineGeek 2.0 beta

A movie and series streaming website.

Features

  • Browse Movies: Explore a vast collection of movies sorted by genre, release date, or popularity.
  • Search Functionality: Easily find movies by title, director, or cast.
  • Recommendation Engine: Get personalized movie recommendations based on your preferences and viewing history.
  • User Authentication: Securely sign in with Google for personalized experiences.
  • Feedback System: Provide feedback through Official Telegram and Twitter Handles.
  • Responsive Design: Enjoy a seamless experience across devices with our responsive web design.
  • Admin Dashboard: Manage users, content, and view.
  • Analytics: Monitor real-time analytics to understand user behavior and optimize content delivery.

Tech Stack

  • Next.js – framework
  • TypeScript – language
  • Tailwind – CSS
  • Vercel – deployments
  • TMDb - movie database
  • Vidsrc.cc - streaming links
  • Flask – web framework
  • SQLAlchemy – SQL toolkit and ORM
  • SQLite – relational database
  • Firebase – backend services (authentication, data storage, etc.)
